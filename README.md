# Tony Knox | Photographer, Artist, Curator

A professional, high-performance portfolio website for Tony Knox, bridging the gap between high-octane ringside wrestling sports photography and the sterile, conceptual spaces of contemporary fine art galleries.

This project was designed and built by **Ben Simpson**.

---

## 📸 About Tony Knox

Based in the North West of England, Tony Knox has spent over two decades capturing the raw intensity and theatrical storytelling of modern professional wrestling while simultaneously curating contemporary art programs, public arts projects, and community exhibitions. He holds a Masters in Fine Art (MFA) from the University of Central Lancashire (2007) and is widely regarded as one of the UK's premier ringside action photographers.

---

## ✨ Features

- **Dynamic Visual Portfolios**:
  - **Ringside Wrestling**: Capturing in-ring collisions, backstage preparation, and historic independent wrestling drama.
  - **Contemporary Curation**: Exhibition curation, site-specific installations, and community canvas programs.
  - **The Mothman**: Tony's performance art documentation and conceptual inquiries.
- **Interactive Story & Timeline**: A curated narrative of Tony's journey, from early analog film experimentation to high-profile curatorial assignments.
- **Live Booking & Inquiry Terminal**: A sleek, user-friendly booking and contact system for galleries, wrestling promotions, and print sales.
- **Modern Performance & Design**:
  - Immersive high-contrast dark visual layout.
  - Rich micro-interactions and smooth transitions powered by Framer Motion.
  - Fully responsive, desktop-first precision architecture built with Tailwind CSS.

## 🚀 Running Locally

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) (v18 or higher) installed on your system.

### Steps

1. **Clone the Repository**:
   ```bash
   git clone <your-repository-url>
   cd <repository-directory>
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start the Development Server**:
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000` or the port displayed in your terminal.

4. **Build for Production**:
   ```bash
   npm run build
   ```
   This will compile the static assets into the `dist/` folder, ready for deployment.
